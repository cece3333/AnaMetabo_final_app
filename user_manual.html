User Manual for AnaMetabo Application

Introduction

AnaMetabo is a user-friendly bioinformatics platform designed to analyze and visualize metabolic networks, with a specific focus on the H5N1 virus. It supports researchers in exploring virus-host dynamics and modeling biological interactions through an interactive web-based interface.

Requirements:

  • Stable internet connection.
  • A modern browser (e.g., Chrome or Firefox).
  • No installation required; runs directly in the browser.

Getting Started

Accessing AnaMetabo

Open AnaMetabo through the provided URL. The home page introduces the application, outlining its features. A sample SBML file is available for download to explore the application.

Steps to Upload Your Data:

  1. Go to the Network Visualization tab.
  2. Click "Import an SBML File" and select your file.
  3. Ensure your file adheres to the SBML standard.

SBML File Components

  • Metabolite: Circular nodes.
  • Enzyme: Diamond-shaped nodes.
  • Reaction: Red square nodes.

Using the Application

Home Page

This section provides an overview of AnaMetabo and includes links to download a sample SBML file and access the user manual.

Network Visualization

After uploading an SBML file, AnaMetabo generates a graphical representation of the network:

  • Visualize nodes (e.g., proteins, metabolites) and edges (e.g., reactions).
  • Adjust graph layouts (Hierarchical, Circular).
  • Interact with the graph:
    • Hover: View node details.
    • Add/Remove: Create or delete nodes and edges.

Advanced Analysis

  • Centrality Analysis: Identify key nodes.
  • Shortest Path: Find efficient node-to-node routes.
  • Clustering: Detect subsystems or node groups.

Results are displayed graphically and in tabular form.

Interactive Features

  • Drag and rearrange nodes for better visualization.
  • Highlight connections by selecting multiple nodes.
  • Customize node colors and sizes.
  • Use "Graph Summary" to view network statistics.

Data Management

  • Upload files in .xml or .sbml format.
  • Export modified graphs in SBML format or as PNG images.
  • Save progress using "Export SBML".

Troubleshooting and FAQs

  • Ensure your SBML file is correctly formatted and under 10 MB.
  • Test with the sample SBML file to verify functionality.
  • For performance issues, reduce network complexity.
